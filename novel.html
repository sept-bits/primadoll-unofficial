Prima Doll Unofficial Translation

This website is dedicated to the unofficial translation of Prima Doll Encore, the Prima Doll web novel.
It uses the same presentation as the original website in order to give you the same reading experience.
Translation is ongoing, so far eleven chapters are available.
You can switch back to the original Japanese text at any time from the menu.
We would be pleased to get feedback from you on Twitter (Sep717) or Discord (Sep7#7617).
